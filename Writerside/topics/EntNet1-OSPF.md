# OSPF

**OSPF** (Open Shortest Path First): Is een **"link-state routing protocol"**.

**Link:** Interface op een router, netwerk segment dat 2 routers verbindt, een netwerk dat aan één router verbonden is...
**Alles wat aan één of meerdere routers verbonden is.**

**Informatie over een link** is de **status** van de **link**.

**OSPF** gebruikt de **statussen** van **links** om de **beste route** te **vinden**.
